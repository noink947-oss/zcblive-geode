# Android Port Documentation

## Project Overview
This repository contains the Android port of the ZCB Live Geode project. This project aims to bring the core functionality of ZCB Live Geode to Android devices, allowing users to experience the features of the app on their mobile devices.

## Features
- **Cross-Platform Compatibility**: Supports various Android devices and screen sizes.
- **User-Friendly Interface**: Designed with a focus on ease of use and accessibility.
- **Performance Optimization**: Enhanced performance tailored for mobile hardware.

## Getting Started
1. **Clone the Repository**:  
   `git clone https://github.com/noink947-oss/zcblive-geode.git`
2. **Open the Project**:  
   Open the cloned repository in Android Studio.
3. **Build the Project**:  
   Run the build process to compile the Android app.
4. **Run on Device**:  
   Connect your Android device and run the app directly from Android Studio.
